# VoroSwarm
**VoroSwarm** is the collision avoidance algorithm for swarming agents based on Voronoi cells

<br/>

## 1. Dependency
```bash
sudo apt-get update
sudo apt-get install libeigen3-dev
sudo apt-get install voro++-dev
```

<br/>

## 2. Example Code
Checkout the folder **voro_swarm_example**
